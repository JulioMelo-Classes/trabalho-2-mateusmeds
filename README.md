## Para compilar

```console
No diretório raiz
make
```

## Para executar
Você pode rodar o sistema e em logo depois digitar os comandos seguidos de ENTER
```console
cd build
./concordo
```

```console
cd build
./concordo < ../data/script_comandos.txt
./concordo < ../data/script_servidores.txt
./concordo < ../data/script_testes.txt
```
